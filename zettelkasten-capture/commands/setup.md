# zettelkasten-capture: Setup

Interactive first-time setup for zettelkasten-capture.

## Steps

1. Check if `~/.claude/zettelkasten-capture/config.sh` already exists
   - If yes: ask "Config already exists. Reconfigure? (y/n)"
   - If no: proceed

2. Ask: "What is the path to your Obsidian vault?"
   - Validate the path exists: `[ -d "$path" ]`
   - If not found: ask to retry or create it
   - Common locations to suggest:
     - `~/Documents/Obsidian/`
     - `~/Library/Mobile Documents/iCloud~md~obsidian/Documents/`

3. Ask: "Inbox folder name within your vault? (default: Inbox)"
   - Create it if it doesn't exist: `mkdir -p "$vault/$inbox"`

4. Ask: "Auto-create excalidraw stubs when diagrams are detected? (y/n, default: y)"

5. Ask: "Auto-push finalized notes to autoimprove-agents KB? (requires autoimprove-agents plugin) (y/n, default: n)"
   - If yes: check if autoimprove-agents is installed at `~/.claude/plugins/local/autoimprove-agents/`
   - If not installed: warn and set to false

6. Create config directory and write config:
   ```bash
   mkdir -p ~/.claude/zettelkasten-capture
   ```
   Write to `~/.claude/zettelkasten-capture/config.sh`:
   ```bash
   # zettelkasten-capture configuration (generated by /zettelkasten-capture:setup)
   export ZC_OBSIDIAN_VAULT="<path>"
   export ZC_INBOX_FOLDER="<folder>"
   export ZC_EXCALIDRAW_THRESHOLD=<0|1>
   export ZC_AUTO_PUSH_TO_KB=<true|false>
   ```

7. Verify: source the config and confirm vault is accessible
8. Show: "Setup complete! Next steps:"
   - "Session drafts will appear in `$vault/$inbox/` after each session"
   - "Run `/zettelkasten-capture:finalize` to promote drafts to permanent notes"
